# Government Timeline — Build Process

This folder contains the working files used to build `government_timeline.json`
(the verified deliverable, documented separately — see source list below).

It is not part of the main parsing pipeline. It's kept for transparency on how
the timeline was constructed, since several steps involved manual data entry
and an external LLM rather than pure code.

## Contents

- `government_timeline_build.ipynb` — the full build process, from raw election
  data and Wikipedia tables through to the final JSON.
- `pm_timeline_working.xlsx` — manually filled in (election dates, own_mandate,
  opposition_party, coalition_parties) using sources listed below.
- `election_data.xlsx` — UK general election results (seats by party, 1910–2024),
  used to fill in seats_won and total_seats.
- `text.txt` — end_reason and notes generated by Gemini, merged into the final
  JSON in the last notebook cell.

- https://en.wikipedia.org/wiki/List_of_prime_ministers_of_the_United_Kingdom

## Note

`end_reason` and `notes` fields were generated by Gemini 3.1 Pro, not derived
directly from the sources above. Treat those two fields with more scrutiny
than the rest of the data (dates, seats, majorities), which came from the
primary sources.

This notebook is not designed to be re-run blindly — see the note at the top
of the notebook itself for where manual steps happen.
